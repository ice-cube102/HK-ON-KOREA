# HKON Website v1.1 Update Log

## 1. Content & Localization Enrichment 🌐
- **Dynamic Localization:** The website language is now automatically detected based on the user's browser language.
- **Enhanced Translations:** All translations across English, Korean, and Chinese (`TRANSLATIONS` object) were significantly enriched for a more professional tone.
- **"About Us" 스케일업:** 단순히 짧은 설명을 넘어서 글로벌 프리미엄 F&B를 선도하는 HKON의 철학과 비전(글로벌 네트워크, 철저한 물류/콜드체인, B2B/B2C 파트너십)을 풍성하게 담아냈습니다.
- **"What's New?" 포커스 이동:** 기업의 일반적인 공지사항에서 나아가 "티라미수 클라시코 단독 런칭" 등 **신제품 소개 및 안내**를 중점적으로 다루도록 개편했습니다.

## 2. 모달 내 제품 UI 파격 개편 ("미디어 자료실 + 공식 페이지") 📺
- **리치 컴포넌트 추가:** '자세히 보기'를 누르면 제품 상세 모달 안에 "브랜드 스토리 영상 시청", "제품 카탈로그 다운로드(미디어 자료실)", "브랜드 담당자 문의" 등의 강력한 CTA(Call-to-Action) 요소가 들어간 프리미엄 모달이 뜨도록 변경했습니다.
- **공식 웹사이트 링크:** 각 글로벌 브랜드(하겐다즈, 란티코 등)의 공식 웹사이트로 아웃링크할 수 있는 요소도 모달 내에 배치하여 연결성을 극대화했습니다.
- **Sparkles & Animations:** 제품 모달이 뜰 때 부드럽게 스태거(Stagger)되는 애니메이션과 아이콘을 배정하여 프리미엄 느낌을 강조했습니다.

## 3. 푸터 소셜 미디어 및 미디어 아이콘 추가
- Footer 영역의 하단(개인정보처리방침 등) 옆에 블로그(Blog), 유튜브(YouTube), 인스타그램(Instagram)으로 이어지는 소셜 아이콘을 우아하게 배치했습니다.
- 모바일과 PC 각각의 레이아웃에서 깨짐 없이 나오도록 반응형으로 대응 완료했습니다.

## 4. UI/UX & Animation Polish ✨
- **Swipeable Hero Interaction:** 메인 배너에서 모바일 친화적인 스와이프 액션 도입.
- **Interactive "Tilt" Effects:** 제품 소개 및 히어로 이미지들에 マ우스 오버 시 입체적인 틸트(Tilt) 애니메이션 탑재.
- **System Theme Adoption:** 기기 다크/라이트 설정에 따른 자동 테마 변화(prefers-color-scheme) 지원.
- **Mobile Header Upgrade:** 모바일에서 접근성 높은 햄버거 메뉴 및 직관적인 테마 변경 토글 적용.
- 원본 이미지 복구: About Us 등의 이미지를 기존 사이트 톤앤매너에 맞는 "그린자이언트(Green Giant)", "란티코 머신" 등의 원본 이미지로 되돌렸습니다.
